Необходимо реализовать логику работы элемента file-uploader. Он должен отвечать следующим требованиям:
1. Выбор файла можно осуществить при помощи нативного окна для выбора файлов, оно должно открываться при клике на кнопку “Or select file to upload”, либо при перетаскивании файла из ОС в область загрузки на сайте.
2. При выборе изображения, оно устанавливается в качестве фона для области загрузки файла.
3. Поддерживаемые форматы изображений: png, jpeg, jpg.
4. При выборе любого другого файла (не изображения), необходимо вывести информацию о названии файла и размере. Информацию нужно разместить под кнопкой “Or select file to upload”.
5. Если дропнуть файл вне контейнера, то ничего не должно происходить (по умолчанию файл откроется в браузере на новой вкладке, этого надо избежать).
